## 📊 Что генерирует программа

### 4 аудиофайла после фильтрации:
- `MUHA_filtered_300_3400.wav` - оптимальный (голос)
- `MUHA_filtered_200_3000.wav` - широкий диапазон
- `MUHA_filtered_500_3000.wav` - узкий диапазон
- `MUHA_filtered_100_1000.wav` - только низкие частоты

### Графики по каждому пункту:
- **Пункт 1.3 (ФНЧ):** временные области, спектры, маски, MSE
- **Пункт 1.5 (Режектор):** влияние ширины, частоты, амплитуды
- **Пункт 1.7 (Комбинированный):** сравнение фильтров, тепловая карта
- **Пункт 1.9 (ФВЧ):** временные области, спектры, MSE
- **Пункт 2 (Аудио):** анализ сигнала, результаты фильтрации

## ⚙️ Настройка параметров

В `config.py` можно изменить:
```python
SignalParams.b = 0.2     # уровень белого шума
SignalParams.c = 0.5     # амплитуда гармонической помехи
SignalParams.d = 20.0    # частота помехи (Гц)
```

## 📝 Важно

- Графики сохраняются автоматически (без показа)
- Все пути формируются относительно корневой директории
